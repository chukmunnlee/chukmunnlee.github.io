Artificial intelligence is a branch of computer science where application learns to function intelligently and independently without human intervention. We hear about AI breakthroughs almost on a daily basis.

To teach these programs to learn and to behave intelligently, a variety of techniques are used including data science and statistics, image processing, natural language processing, pattern recognition, neural networks, feedback and rewards.

Job Roles for Artificial Intelligence Track

AI Systems Engineer

Creates technology architectures and systems that integrate AI capabilities into new or existing products. For example, integrating and deploying algorithms to predict customer spending at an e-commerce website like Lazada.

    Python for Data, Ops and Things Data and Feature Engineering for Machine Learning Supervised and Unsupervised Modeling with Machine Learning Containers for Deployment and Scaling Apps

AI Data Engineer

Sources and manages the data pipeline for training AI models. This role determines what data sources are relevant for solving a business problem using AI, and applies domain expertise to transform data into features. For example, cleaning, normalising and aggregating factory equipment sensor data to be used for predictive maintenance.

    Python for Data, Ops and Things Data and Feature Engineering for Machine Learning Supervised and Unsupervised Modeling with Machine Learning Feature Extraction and Supervised Modeling with Deep Learning

AI Data Scientist

Applies, trains and fine-tunes AI algorithms to solve challenging business problems using data. For example, training a deep neural network to predict stock market portfolio performance based on socio-economic indicators.

    Python for Data, Ops and Things Data and Feature Engineering for Machine Learning Supervised and Unsupervised Modeling with Machine Learning Feature Extraction and Supervised Modeling with Deep Learning Sequence Modeling with Deep Learning

DevOps is a set of software development practices that combines software development (Dev) and system administration (Ops). The objective of DevOps is to shorten the development/deployment cycle so that companies can stay competitive by bringing new products and features to the market.

DevOps achieves this productivity gain by automating test, workflow and infrastructure. Products are continuously improved through monitoring.

Job Roles for DevOps Track

Frontend/Client-Side Developer

Produce slick, responsive HTML5 applications with HTML, CSS, JavaScript/TypeScript and frameworks like Angular. A client-side developer can also branch into other fields like games levering popular HTML5 game engines like Phaser. Some examples of single page HTML5 applications include Google Docs or web games like those on itch.io.

    Client Side Foundation Containers for Deployment and Scaling Apps

Mobile (Hybrid) developer

A hybrid mobile application developer is a person who writes applications for Android and iOS using cross platform technologies such as Progressive Web App, Cordova or NativeScript. Examples of NativeScript and PWAs include Sennheiser Smart Control and AliExpress.

    Client Side Foundation Containers for Deployment and Scaling Apps

Back-end/Server-Side Developer

Writes scalable applications that runs on servers. These applications handle the business logic, analytics, and CRUD operations to records held in SQL and NoSQL databases on behalf of client requests.

    Server Side Foundation Persistence and Analytics Fundamentals Security, Notification and Messaging Fundamentals Containers for Deployment and Scaling Apps

Full Stack Developer

A developer who is capabable of developing both the client and server software. They are developers who are comfortable with using HTML, CSS, JavaScript to develop HTML5 applications; they are also adept at developing on backend platform like Express and performing CRUD operations on database.

Solution Architect

Responsible for defining and delivering a robust end-to-end system architecture.
